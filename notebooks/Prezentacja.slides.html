Struktura warstwowa¶

Cebula ma warstwy - Nanoczstki maj warstwy ... i nie s jak tort!

Aby uzyska zamierzon struktur warstwow wykorzystamy t sam funkcj aby zidentyfikowa kolejne warstwy czstki i zdefiniowa ich skad - na przemian Fe i Pt.

In [3]:
def LC_ico(a1,a2,ln):
    '''
    Stwrz struktur warstwow rzdu ln 
    o naprzemiennych warstwach atomw a1, a2
    '''
    ico=Icosahedron(a1,ln)
    icon=ico.get_atomic_numbers()
    for l in range(ln,0,-1):
        if (ln-l)%2 :
            a=a1
        else :
            a=a2
        il=Icosahedron(a,l)
        an=il.get_atomic_numbers()
        icon[:len(an)]=an
    ico.set_atomic_numbers(icon)
    return ico

Dynamika molekularna - co to jest?¶

Dynamika molekularna to badanie ukadw atomowych przez numeryczne rozwizanie ich rwnania ruchu a nastpnie analiz tak uzyskanej trajektorii.

Rwnanie ruchu atomu o indeksie $i$:

$$ \frac{d^2{\mathbf x}_i}{{dt}^2}=\frac{{\mathbf F}_i}{m_i} $$

rozwizywane jest klasycznymi metodami numerycznego cakowania rwnan ruchu (np. algorytm Verleta), siy ${\mathbf F}_i$ pochodz z rachunku ab-initio (siy Helmana-Feynmana).

Dla naszej struktury przeprowadzamy seri oblicze w ktrych nadajemy atomom rn redni energi kinetyczn - czyli temperatur:

$$ T=\frac{2\langle E_k \rangle}{3k_B}. $$

W rezultacie otrzymujemy trajektorie ukadu, ktre poddajemy dalszej analizie.

Analiza trajektorii¶

W materiaach wiczeniowych znajduje si kilka dalszych przykadw analizy uzyskanych trajektorii. Tutaj zajmiemy si uzyskaniem wykresu kalorymetrycznego ($E_p - E_k$ jako funkcja temperatury $T$), ktry pozwoli nam rozpozna temperatur topnienia nanoczstki o platynowej powierzchni.

  • Enargia potencjalna wyliczana jest przez program ab-initio i zawarta jest we wczytanych danych
  • Energi kinetyczn musimy wyliczy na podstawie trajektorii
$$ E_k(t_i)=\frac{1}{N_{at}}\sum_{n=1}^{N_{at}}\frac{m_n {\mathbf v_n}(t_{i})^2}{2}=\frac{1}{N_{at}}\sum_{n=1}^{N_{at}}\frac{m_n ({\mathbf x_n}(t_{i+1})-{\mathbf x_n}(t_{i}))^2}{2} $$

def calc_velocities(a1,a2,dt):
    '''
    Wyliczenie prdkoci atomw na podstawie pozycji a1, a2
    oraz odstpu czasu dt
    '''
    dx=(a2.get_positions()-a1.get_positions())
    c=Atoms(a2)
    tv=sum(c.get_cell(),axis=0)/2
    c.set_positions(dx)
    c.translate(tv)
    c.set_scaled_positions(c.get_scaled_positions())
    c.translate(-tv)
    return c.get_positions()/dt

def calc_kinen(tr,dt):
    '''
    Wyliczenie energii kinetycznej atomw w trajektorii tr 
    przy kroku czasowym dt.
    '''
    m=tr[0].get_masses()
    Ek=[m*sum(calc_velocities(a1,a2,dt)**2,axis=1)/2 
                    for a1,a2 in zip(tr[:-1],tr[1:])]
    return Ek
    
def calc_temp(tr,dt):
    '''
    Wyliczenie chwilowej temperatury w trajektorii tr
    przy kroku czasowym dt
    '''
    eka=calc_kinen(tr,dt)
    ek=sum(eka,axis=-1)
    ekd=std(eka,axis=-1)
    n=len(tr[0].get_masses())
    c=n/(n-1)
    return 2*c*ek/(3*n*un.kB), 2*c*ekd/(3*un.kB)

Krzywa kalorymetryczna¶

Wyliczamy:

  • Energi potencjaln (zapisana w rezultatach oblicze ab-initio)
  • Energi kinetyczn
  • redni temperatur
In [9]:
dat=[]
for T in sorted(md):
    dt,tr=md[T]
    epot=array([s.get_potential_energy() for s in tr])
    epot=((epot[:-1]+epot[1:])/2)
    ekin=sum(array(calc_kinen(tr,dt*un.fs)),axis=-1)
    etot=epot+sum(ekin,axis=-1)
    temp_v, temp_d=calc_temp(tr,dt*un.fs)
    atemp=mean(temp_v)
    dat.append([atemp, mean(epot), mean(ekin),
                mean((etot-mean(etot))**2)/(un.kB*(atemp**2))])
dat=array(dat).T
